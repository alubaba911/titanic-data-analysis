Titanic Data Analysis

Ce projet présente une analyse exploratoire du dataset Titanic avec Python.

## Résumé des résultats

- Taux de survie global : environ 38 %.
- Facteur le plus déterminant : le sexe.
- Facteur le moins déterminant : certaines variables comme la cabine.
- Observation intéressante : les passagers de 1ère classe survivent beaucoup plus souvent.
- Limitation : plusieurs valeurs manquantes dans les données (âge, cabine).



Exploratory data analysis of the Titanic dataset using Python.

Tools:
- pandas
- matplotlib
- seaborn
- Google Colab

Main steps:
- Data cleaning
- Descriptive statistics
- Conditional probabilities
- Correlation analysis
- Feature engineering (FamilySize)
